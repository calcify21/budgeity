# 🚀 Budgeity Commands Guide

- **`npm run dev`**: Start UI Dev server
- **`npm run dev:emu`**: Start UI Dev Server & Emulators
- **`npm run emu`**: Start Firebase emulators
- **`npm run preview`**: Preview production build
- **`npm run tran:sync`**: Generate i18n types
- **`npm run tran:check`**: Audit translation keys
- **`npm run tran:clean`**: Clean unused translations
- **`npx tsc --noEmit`**: Typecheck the codebase
- **`npm run lint`**: Stylecheck the codebase
- **`npm run build:f`**: Build for Firebase
- **`npm run build:h`**: Build for Hostinger
- **`npm run deploy`**: Builds and deploy the app to Firebase
- **`npm run deploy:r`**: Deploy Firestore rules
- **`npm run depcheck`**: Scans your code files for unused packages (not in package.json)
- **`npm install`**: Install all project dependencies listed in package.json
- **`npm prune`**: Deletes any package files in your node_modules folder that are not listed in package.json
- **`npm cache clean --force`**: Completely empties the local cache folder where npm stores downloaded packages.
